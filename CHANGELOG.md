# CHANGELOG

## Ver1.1

### 変更点

- レビュー画面の動画数別レイアウトを固定ルールに修正。1本は1列、2本は2列、3本は3列、4本は2×2、15本は5列×3行で表示。
- レイアウト計算対象を現在表示中の有効動画だけに限定。非表示化・destroy済みの動画は除外。
- グリッド列をJavaScriptから `repeat(columns, minmax(0, 1fr))` で設定し、タイルをセル内で16:9最大表示するよう調整。
- `tileWidth` / `tileHeight` をJavaScriptで計算し、グリッドと各動画カードへ明示サイズとして反映。
- YouTube iframeの親要素がカード全体に広がるようCSSを補強。
- 共有URLを開くとレビュー画面へ直行するフローを実装。
- 共有URLを `#r=` の短い独自形式へ変更。
- payloadを `encodeURIComponent()` / `decodeURIComponent()` で扱うよう変更。
- 共有URLにはVideo IDと開始位置だけを含める仕様へ変更。
- プレイヤー名をVer1.1の通常フローから削除。
- 設定画面を共有URL作成専用に整理。
- 設定画面の入力途中保存を `localStorage` から `sessionStorage` へ変更。
- 共有URLから読み込んだ設定を自動保存しないよう変更。
- JSON機能を詳細機能として折りたたみ表示へ変更。
- レビュー画面の操作バーを下部固定、自動非表示へ変更。
- 30秒戻す/30秒進めるを追加。
- reviewTimeをマスタープレイヤー基準で更新するよう変更。
- 初期音声を全ミュートへ変更。
- 音声ONは1本のみになるよう整理。
- 拡大した動画が音声ONになるよう変更。
- 正常動画/エラー動画の一時非表示を追加。
- 非表示動画を再生操作・再同期・レイアウト計算の対象外に変更。
- エラー動画に「YouTubeで開く」を表示。
- 共有URL破損、有効動画0本、一部破損、開始位置破損の処理を追加。
- エラー153対策の `referrer` / `origin` 方針を維持。

### 削除

- 画質制御機能の仕様を削除。
- 画質希望コードを共有URL仕様から除外。
- ツール側から画質を一括指定・強制する方針を削除。

### 既知の制約

- YouTubeの仕様上、完全なフレーム単位同期は保証しない。
- `file://` 直開きではエラー153が出る可能性がある。
- 15本同時再生はPC、GPU、メモリ、回線、ブラウザ状態に強く依存する。
- ショートカットはYouTube iframeにフォーカスがあると効かない場合がある。
- 非表示にした動画を戻すにはページ再読み込みが必要。
- リーダー操作のリアルタイム同期は未実装。
