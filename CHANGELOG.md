# CHANGELOG

## Ver1 - 初回実装

### 実装済み

- 新規静的Webアプリとして `index.html` / `style.css` / `app.js` を作成。
- 最大15本のYouTube動画枠を表示。
- プレイヤー名、YouTube URL、個別開始位置の入力に対応。
- `watch?v=`、`youtu.be`、`embed`、`live`、動画ID単体からの動画ID抽出に対応。
- `HH:MM:SS`、`MM:SS`、秒数形式の開始位置入力に対応。
- 不正な開始位置を0秒扱いにし、UIで警告表示。
- YouTube IFrame Player APIによる埋め込み再生。
- 反省会タイム基準の同期。
- 全再生、全停止、5秒/10秒戻す、5秒/10秒進める、任意ジャンプ、再同期を実装。
- 倍速一括変更を実装。
- 個別ミュート、全ミュート、全ミュート解除、選択動画だけ音声ONを実装。
- 拡大表示モーダルを実装。
- 各枠の開始位置、実再生位置、ミュート状態、読み込み状態、エラー状態を表示。
- アプリ全体の反省会タイム、再生状態、倍速、読み込み済み動画数を表示。
- localStorageへの設定保存と復元を実装。
- JSONエクスポート/インポートを実装。
- URL hashを使った共有リンク生成と復元を実装。
- 5列×3行を基本にしたレスポンシブレイアウトを実装。
- 注意書き、README、SPECを整備。
