# CHANGELOG

## Ver1 - 初回実装

### 実装済み

- 新規静的Webアプリとして `index.html` / `style.css` / `app.js` を作成。
- 最大15本のYouTube動画枠を表示。
- プレイヤー名、YouTube URL、個別開始位置の入力に対応。
- `watch?v=`、`youtu.be`、`embed`、`live`、動画ID単体からの動画ID抽出に対応。
- `HH:MM:SS`、`MM:SS`、秒数形式の開始位置入力に対応。
- 不正な開始位置を0秒扱いにし、UIで警告表示。
- YouTube IFrame Player APIによる埋め込み再生。
- 反省会タイム基準の同期。
- 全再生、全停止、5秒/10秒戻す、5秒/10秒進める、任意ジャンプ、再同期を実装。
- 倍速一括変更を実装。
- 個別ミュート、全ミュート、全ミュート解除、選択動画だけ音声ONを実装。
- 拡大表示モーダルを実装。
- 各枠の開始位置、実再生位置、ミュート状態、読み込み状態、エラー状態を表示。
- アプリ全体の反省会タイム、再生状態、倍速、読み込み済み動画数を表示。
- localStorageへの設定保存と復元を実装。
- JSONエクスポート/インポートを実装。
- URL hashを使った共有リンク生成と復元を実装。
- 5列×3行を基本にしたレスポンシブレイアウトを実装。
- 注意書き、README、SPECを整備。

### 未実装

- ホスト操作を参加者ブラウザへリアルタイム同期する機能。
- 外部データベース保存。
- ユーザー認証。
- YouTube APIキーを使う詳細メタデータ取得。
- フレーム単位の完全同期保証。

### 次に改善すべき内容

- リアルタイム同期用の部屋機能。
- タイムラインメモ、クリップポイント、指摘コメント。

## Ver1 改修

### 実装済み

- YouTube読み込みエラー153対策として `meta name="referrer"` を追加。
- YouTube Player作成時に `enablejsapi` と、可能な場合の `origin` 指定を追加。
- エラー153/100/101/150の説明を分かりやすく更新。
- 設定モードとレビューモードを追加。
- レビューモードでは入力欄、設定パネル、空欄枠を非表示化。
- 有効な動画だけをレビューグリッドへ表示。
- 有効動画数と画面サイズから16:9タイルの最適列数を計算する自動レイアウトを実装。
- レビューモード用の固定操作バーを追加。
- 操作バーの表示/非表示切替とマウス移動時の一時表示を追加。
- レビューモードでクリック拡大、Esc解除、選択動画だけ音声ONを追加。
- Space、ArrowLeft、ArrowRight、R、Escのキーボードショートカットを追加。
- READMEにlocalhost起動、公開URL利用、エラー153対処を追記。

### 残っている注意点

- `file://` で直接開いた場合、ブラウザやYouTube側の仕様によりエラー153が出る可能性がある。
- YouTubeの完全なフレーム単位同期は保証できない。
- 15本同時再生はPC負荷が高い。
